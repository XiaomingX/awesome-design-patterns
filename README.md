# awesome-design-patterns
## 优秀的软件和架构设计模式精选指南

> 精心整理的软件与架构相关设计模式集合，涵盖主流编程语言、架构方法和应用场景。

[软件设计模式](https://en.wikipedia.org/wiki/Software_design_pattern) - 一种通用的、可复用的解决方案，用于特定上下文中的常见问题。它是一种描述或模板，可以在不同情况下用来解决问题。
